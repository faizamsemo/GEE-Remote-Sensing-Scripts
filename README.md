# GEE-Remote-Sensing-Scripts
This repository contains Google Earth Engine scripts for GIS, remote sensing, hydrology, climate, and environmental monitoring.

## Main Topics

- Water Quality Monitoring
- Crop NPP Analysis
- Drought Risk Mapping
- Land Surface Temperature
- Temperature and Precipitation Analysis
- Land Use and Land Cover Classification

## Tools Used

- Google Earth Engine
- JavaScript
- MODIS
- Sentinel-2
- Landsat
- CHIRPS
- ERA5-Land
